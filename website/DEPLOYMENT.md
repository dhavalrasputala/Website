## Quick publishing options

### Option 1: Vercel
1. Open https://vercel.com
2. Sign in and import this project
3. Use these settings:
   - Build command: npm run build
   - Output directory: build
4. Deploy

### Option 2: Netlify
1. Open https://netlify.com
2. Drag and drop the build folder or connect the GitHub repo
3. Use these settings:
   - Build command: npm run build
   - Publish directory: build
4. Deploy
